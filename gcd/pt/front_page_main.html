Novidades no site!

Criamos algumas novas funcionalidades para o site recentemente. Muitas delas utilizam os nossos dados em "link", por exemplo criador, protagonista e personagem, para os quais a migração é um projeto em andamento. Em média, conseguimos migrar quase metade dos nossos dados do formato "texto" para "link".

Por exemplo, temos listas de capas para

  • criadores (p.ex. Kaare Andrews)
  • features (p.ex. Star Wars) and
  • characters (p.ex. The Shadow).

Para criadores, já tínhamos há algum tempo listas dos seus trabalhos por histórias, edições, séries ou protagonistas. Recentemente, criamos listas para cocriadores (p.ex. George Pérez). Também conseguimos mostrar para protagonistas (p.ex. Iron Man) os criadores, assim como para os personagens (p.ex. Wolverine).

Recentemente, criamos outros tipos de listas, como por exemplo uma que mostra as edições com suas capas e principal história (com mais páginas). Ela pode ser usada para

  • séries (p.ex. Superamigos) or
  • editoras (p.ex. Editora Abril).
Estas listas de edições com as capas e histórias mais longas podem também ser usadas para as listas de lançamentos mensais e semanais.

Para navegação, adicionamos em muitas das páginas de listas uma filtragem adicional por país, idioma ou editor. Assim, você pode ver facilmente quais edições em português registramos para o Jack Kirby.

Para todas essas listas, é claro, mais dados precisam ainda ser inseridos ou migrados das entradas em texto para registros em link. Se você já pensou em ajudar com o conteúdo do banco de dados, agora é uma boa hora.
