Como Ajudar

Você pode nos ajudar de diversas maneiras para melhorar o site e seu conteúdo.
  • Você pode inserir dados que estão faltando, atualizar os que já existem ou fazer upload de capas digitalizadas. Primeiro, crie uma conta e daí você pode começar a contribuir.
  • Fazer doações para ajudar nos custos operacionais, por exemplo com infraestrutura de servidores. Somos uma organização sem fins lucrativos e quaisquer fundos serão utilizados para nosso objetivo de documentar e indexar todas as revistas em quadrinhos.
  • Precisamos de web designers e programadores voluntários! Por favor, entre em contato pelo nosso grupo de discussão de tecnologia, o gcd-tech group ou visite nossa página de documentação técnica se você puder ajudar em alguma destas funções:
    • Web designer / front-end developer (HTML / CSS / JavaScript)
    • programação em Python / Django
    • servidor de busca ElasticSearch
    • Web Services API
    • Database Performance (MySQL)
